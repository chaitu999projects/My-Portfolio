What i do

Responsive design

Responsive Web Development is the practice of designing and coding websites that adapt seamlessly to different screen sizes and devices. The goal is to ensure an optimal user experience across desktops, tablets, and mobile phones without requiring separate designs for each device.

Web Development

Web development refers to the process of creating, designing, and maintaining websites and web applications. It involves various technologies, frameworks, and programming languages to ensure a functional, interactive, and visually appealing online presence.

Git & GitHub

Git – A Distributed Version Control System Git is an open-source version control system that helps developers track changes in their code, collaborate efficiently, and manage different versions of a project.

Clean code

Clean code is a coding practice that emphasizes writing clear, efficient, and maintainable code. It ensures that software is easy to understand, modify, and debug, making development smoother and reducing technical debt.

Affiliate Marketing

Affiliate marketing is a performance-based online marketing strategy where individuals (affiliates) earn a commission by promoting products or services of other companies. It is one of the most popular ways to make passive income online.

Online Adds

Online advertising is a digital marketing strategy where businesses and marketers run paid ads on platforms like Facebook, Google, Instagram, and YouTube to reach targeted audiences and drive sales.
